Temperature Conversion

This Ruby program will prompt the user for a temperature in Celsius and let the user know what the corresponding temperature is in Farhenheit. 


def celsius_to_f(celsius)
   (celsius * 1.8) + 32
end

puts "What is the temperature in Celsius?"
input = gets.to_f

val = celsius_to_f(input)

puts "The temperature in Farenheit is #{val} degrees."

puts "Goodbye"
